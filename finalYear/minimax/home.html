Naail's Final Year Project

Project Purpose

The purpose of this final year project is to effectively answer the following questions.

  • Why has gaming been such a popular domain for AI research for so many years?
  • What real world uses can gaming AI's be used for?
  • What steps can be made to create some level of general level AI in the future?

Below are some AI algorithm implementations for you to play around with. We have used several different algorithms,
each being implemented onto a popular game. Each section explains the code implementation in greater detail.
